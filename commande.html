-type Recherche par type de fichier.

-size Recherche par taille de fichier.

GREP
-i Ignorer la casse (ie majuscules, lettres minuscules).
-v Retour toutes les lignes qui ne correspondent pas au modèle.
-w Sélectionnez uniquement les matchs qui forment des mots entiers.
-c Imprimer un nombre de lignes correspondant. Peut être combiné avec l'option -v pour imprimer un nombre de lignes de matchine non
-l Imprimer le nom de chaque fichier qui contient un match. Normalement utilisé lorsque grep est invoqué avec wildcards pour l'argument de fichier .
-n Imprimer le numéro de la ligne avant chaque ligne qui correspond.
-r Récursif, lire tous les fichiers dans le répertoire et les sous-répertoires donné.
-m sortie séparées par des virgules
